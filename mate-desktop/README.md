# MATE Desktop snap

This project attempts to create a working snap of the complete MATE Desktop
environment. The initial objective is to get just the main applications
running.

## Current state

The components are listed in build order.

Bits that build:

  * mate-common
  * mate-desktop
  * mate-user-guide
